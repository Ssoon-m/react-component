# 버튼

## 버튼이 필요한 요소에 사용

특정 사용자에게 액션이나 ~~

참고 사이트

- https://storybook.js.org/tutorials/design-systems-for-developers/react/ko/document/
- https://storybook.js.org/docs/react/api/mdx

export const Template = (args) => <Button {...args} />;

### Button Size

세가지의 크기가 지원됩니다.

<Story name='sizes'>
  <div className='flex'>
    <Button size='small'>Small</Button>
    <Button size='medium'>Medium</Button>
    <Button size='large'>Large</Button>
  </div>
</Story>

### Button Color

두가지 색상이 지원됩니다.

<Story name='color'>
  <div className='flex'>
    <Button size='small' color='white'>
      Small
    </Button>
    <Button size='medium' color='black'>
      Medium
    </Button>
  </div>
</Story>

### Button Color

버튼 비활성화

<Story name='button disabled'>
  <div className='flex'>
    <Button size='small' color='white' disabled>
      Small
    </Button>
    <Button size='medium' color='black' disabled>
      Medium
    </Button>
  </div>
</Story>
